#### **Kisan-Sathi application**



This application is designed to support farmers with smart, data-driven recommendations. It assists users with crop planning, soil analysis insights, market trends, and weather-based guidance. Farmers can also upload reports, images, and other files for more accurate AI-powered suggestions.



The app supports both Hindi and English output, allowing farmers to receive responses in the language they prefer.



###### **Features**

* AI-based recommendations for crops, soil, weather, and market conditions
* Report and image upload for detailed analysis
* Multi-language support (Hindi \& English output)
* Simple and intuitive interface suitable for farmers
* Smart Farming–oriented suggestions to improve productivity and decision-making





###### **Getting Started**



**1. Install dependencies**

Make sure you are inside your project folder and run:

pip install -r requirements.txt

Or use -> "pip install Flask Flask-SQLAlchemy SQLAlchemy python-dotenv requests google-generativeai"



**2. Start the application**

Run the following:

python run.py
